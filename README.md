## 소개

2024년 야구 붐이 오면서 인스타 릴스, 유튜브 쇼츠 등에 중요 장면 영상을 올리는 계정이 기하급수적으로 늘어났습니다. 하지만 하루에 5경기, 한 경기당 최소 3시간씩은 걸리는 야구 경기를 모두 시청하고 중요한 장면을 선정하는 과정은 길고 복잡합니다.  
여기에 시간이 없는 현대인을 위해서 자동으로 하이라이트 장면을 추출해주는 모델을 준비했습니다.
